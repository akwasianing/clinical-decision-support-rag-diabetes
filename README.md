# Clinical Decision Support Assistant Using RAG for Type 2 Diabetes Management

## Project Overview

This project develops a Retrieval-Augmented Generation (RAG) system that answers Type 2 Diabetes management questions using the American Diabetes Association (ADA) Standards of Care in Diabetes—2026 guidelines. The system will retrieve relevant guideline sections and generate evidence-based responses with source citations.

## Data Source

American Diabetes Association (ADA) Standards of Care in Diabetes—2026.

## Repository Structure

* data/ : ADA guideline documents
* notebooks/ : Jupyter notebooks used for development and experimentation
* outputs/ : Evaluation results, outputs, and project artifacts
